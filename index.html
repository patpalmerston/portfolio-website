Coach Me

The Coach Me App was developed for Medical professionals who actively work in preventative care for disadvantaged middle aged women and seniors. As part of the Web Dev Team I had the pleasure of working on the Client side Back End, as well as implementing code standards/set up for the FE React/Redux state management system. I actively worked with the Airtable api to help create user friendly presentations of a clients health metric data. For a more in depth look at the project feel free to check the github repository for the application.

DevDesk Que

The DevDesk Que App was created to help organize and control the flow of student ticket requests for Instructors and Admins. This would allow for a more timely, specific and organize response to student needs. During this project I spear headed the use of the Context Api State management strategy, as well as building out all the features for the Admin side of the DevDesk FE application. Here is access to the github repository for this project

Card Boss

The Card Boss App has a QrScanner implemented for easy access for organizing the various interactions with business professionals when trading contact information. Providing a platform from which you can scan business cards and organize them in your own data base. For this application I implemented the React/Redux state management system, styled the app using Bootstrap and implemented the QrScanner tools to give mobile devices access to updating the sites data base with a quick scan. Here is access to the github repository for this project
